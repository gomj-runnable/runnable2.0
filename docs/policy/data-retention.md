# 데이터 보존 및 삭제 정책

> **[초안]** 본 문서는 서비스 출시 전 법률 전문가 검토가 필요한 초안입니다.

**시행일: 미정**
**버전: 0.1 (초안)**

---

## 1. 계정 데이터

| 데이터 | 보존 기간 | 삭제 조건 |
|--------|----------|----------|
| 이메일, 닉네임 | 회원 탈퇴 후 즉시 삭제 | 탈퇴 요청 처리 후 |
| 비밀번호 해시 | 회원 탈퇴 후 즉시 삭제 | 탈퇴 요청 처리 후 |
| 세션 토큰 | 세션 만료(7일) 후 삭제 | 만료 또는 로그아웃 시 |

## 2. 경로 데이터

| 데이터 | 보존 기간 | 삭제 조건 |
|--------|----------|----------|
| 개인 경로 (비공개) | 회원 탈퇴 후 즉시 삭제 | 탈퇴 또는 개별 삭제 요청 |
| 공개 경로 | 회원 탈퇴 후 **30일 유예** 후 삭제 | 유예 기간 만료 시 |
| GPX 파일 (다운로드 이력) | 보존하지 않음 | 다운로드 후 서버에 저장하지 않음 |

> **공개 경로 30일 유예 이유:** 다른 이용자가 공개 경로를 탐색·저장·공유한 경우를 고려해 즉시 삭제 대신 유예 기간을 둡니다. 유예 기간 중 탈퇴 이용자의 닉네임은 "삭제된 사용자"로 표시됩니다.

## 3. 위치 정보

| 데이터 | 보존 기간 | 비고 |
|--------|----------|------|
| 현재 위치 (GPS 좌표) | 서버에 저장하지 않음 | 클라이언트 측에서만 처리 |
| 경로 좌표 (저장된 경로) | 경로 삭제 시 함께 삭제 | 경로 데이터에 포함 |

## 4. 서비스 이용 기록

| 데이터 | 보존 기간 | 비고 |
|--------|----------|------|
| 접근 로그 (IP, 시간) | 3개월 | 보안 및 어뷰징 방지 목적 |
| API 요청 로그 | 1개월 | 서비스 안정성 모니터링 |
| 오류 로그 | 1개월 | 서비스 개선 목적 |

## 5. 법령에 따른 의무 보존

전자상거래 등에서의 소비자보호에 관한 법률 등에 따라 다음 데이터는 의무 보존합니다.

| 데이터 | 보존 기간 | 근거 법령 |
|--------|----------|----------|
| 결제·계약 기록 | 5년 | 전자상거래법 제6조 |
| 소비자 불만·분쟁 기록 | 3년 | 전자상거래법 제6조 |
| 로그인 기록 | 3개월 | 통신비밀보호법 (검토 필요) |

## 6. 데이터 삭제 요청

이용자는 언제든지 계정 설정 페이지 또는 서비스 문의를 통해 본인의 데이터 삭제를 요청할 수 있습니다.

### 삭제 처리 절차

1. 이용자가 삭제 요청 제출
2. 본인 확인 (이메일 인증)
3. **즉시 삭제:** 계정 정보, 비공개 경로, 세션
4. **30일 후 삭제:** 공개 경로 (삭제 예정 표시)
5. 삭제 완료 이메일 발송

### 처리 기간

- 일반 삭제 요청: **영업일 기준 3일 이내** 처리
- 긴급 삭제 요청 (개인정보 침해 우려): **24시간 이내** 처리

## 7. 서비스 종료 시 데이터 처리

서비스 종료 시 이용자에게 최소 30일 전 공지하며, 다음 절차를 따릅니다.

1. 이용자에게 데이터 내보내기(GPX) 기간 제공
2. 공지 기간 종료 후 모든 개인 데이터 삭제
3. 익명화된 집계 통계 데이터만 보존 가능

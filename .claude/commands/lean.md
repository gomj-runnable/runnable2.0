# Lean Mode

토큰 사용량을 최소화하는 프로젝트 전용 명령이다.

## 동작 규칙

- 사용자가 직접 지정한 파일과 현재 작업에 직접 연결된 파일만 읽는다.
- 넓은 `rg` 검색, 긴 로그 출력, 전체 파일 덤프는 피한다.
- 파일을 읽기 전에 필요한 이유를 한 줄로 정리하고, 대체 가능한 경우 요약으로 충분한지 먼저 판단한다.
- `@`로 첨부되지 않은 파일을 새로 읽어야 하면 사용자에게 먼저 권한을 요청한다.
- 응답은 짧은 문단 위주로 작성하고, 파일별 나열보다 결과와 다음 행동만 남긴다.
- 구현이 가능한 경우 분석보다 수정과 검증을 우선한다.
- `TODO:` 또는 `TODO 0.` 은 구현 대상으로 보고, `TODO`, `TODO-` 등은 제외한다.
- `app/composables/`에서는 추상 타입의 구현체를 `*Impl`로 분리하고, 파라미터와 반환값 계약은 사용자 승인 없이 바꾸지 않는다.
